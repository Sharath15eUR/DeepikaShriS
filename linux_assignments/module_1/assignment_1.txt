Basic Commands in Linux


1) Commands are actually files containing programs, often written in C. How will you find out in which directory does  the file corresponding to the man command resides?

The directory where the file corresponding to the man command resides can be found by using "which" command.
Running this command will display the full path to the man command executable, which will indicate the directory where it is located.

2) How will you find out what is the use of the ps command.

'man ' command can be used to find out the use of any command,not only ps command.
On running this command in shell,it will display the manual page for ps command.
